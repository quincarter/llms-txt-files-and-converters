# Jellyfin API
Description: 

## GET /System/ActivityLog/Entries
**Summary**: Gets activity log entries.
**Parameters**:
- `startIndex` (query integer): Optional. The record index to start at. All items with a lower index will be dropped from the results.
- `limit` (query integer): Optional. The maximum number of records to return.
- `minDate` (query string): Optional. The minimum date. Format = ISO.
- `hasUserId` (query boolean): Optional. Filter log entries if it has user id, or not.

## GET /Auth/Keys
**Summary**: Get all keys.

## POST /Auth/Keys
**Summary**: Create a new api key.
**Parameters**:
- `app`* (query string): Name of the app using the authentication key.

## DELETE /Auth/Keys/{key}
**Summary**: Remove an api key.
**Parameters**:
- `key`* (path string): The access token to delete.

## GET /Artists
**Summary**: Gets all artists from a given item, folder, or the entire library.
**Parameters**:
- `minCommunityRating` (query number): Optional filter by minimum community rating.
- `startIndex` (query integer): Optional. The record index to start at. All items with a lower index will be dropped from the results.
- `limit` (query integer): Optional. The maximum number of records to return.
- `searchTerm` (query string): Optional. Search term.
- `parentId` (query string): Specify this to localize the search to a specific item or folder. Omit to use the root.
- `fields` (query array): Optional. Specify additional fields of information to return in the output.
- `excludeItemTypes` (query array): Optional. If specified, results will be filtered out based on item type. This allows multiple, comma delimited.
- `includeItemTypes` (query array): Optional. If specified, results will be filtered based on item type. This allows multiple, comma delimited.
- `filters` (query array): Optional. Specify additional filters to apply.
- `isFavorite` (query boolean): Optional filter by items that are marked as favorite, or not.
- `mediaTypes` (query array): Optional filter by MediaType. Allows multiple, comma delimited.
- `genres` (query array): Optional. If specified, results will be filtered based on genre. This allows multiple, pipe delimited.
- `genreIds` (query array): Optional. If specified, results will be filtered based on genre id. This allows multiple, pipe delimited.
- `officialRatings` (query array): Optional. If specified, results will be filtered based on OfficialRating. This allows multiple, pipe delimited.
- `tags` (query array): Optional. If specified, results will be filtered based on tag. This allows multiple, pipe delimited.
- `years` (query array): Optional. If specified, results will be filtered based on production year. This allows multiple, comma delimited.
- `enableUserData` (query boolean): Optional, include user data.
- `imageTypeLimit` (query integer): Optional, the max number of images to return, per image type.
- `enableImageTypes` (query array): Optional. The image types to include in the output.
- `person` (query string): Optional. If specified, results will be filtered to include only those containing the specified person.
- `personIds` (query array): Optional. If specified, results will be filtered to include only those containing the specified person ids.
- `personTypes` (query array): Optional. If specified, along with Person, results will be filtered to include only those containing the specified person and PersonType. Allows multiple, comma-delimited.
- `studios` (query array): Optional. If specified, results will be filtered based on studio. This allows multiple, pipe delimited.
- `studioIds` (query array): Optional. If specified, results will be filtered based on studio id. This allows multiple, pipe delimited.
- `userId` (query string): User id.
- `nameStartsWithOrGreater` (query string): Optional filter by items whose name is sorted equally or greater than a given input string.
- `nameStartsWith` (query string): Optional filter by items whose name is sorted equally than a given input string.
- `nameLessThan` (query string): Optional filter by items whose name is equally or lesser than a given input string.
- `sortBy` (query array): Optional. Specify one or more sort orders, comma delimited.
- `sortOrder` (query array): Sort Order - Ascending,Descending.
- `enableImages` (query boolean): Optional, include image information in output.
- `enableTotalRecordCount` (query boolean): Total record count.

## GET /Artists/{name}
**Summary**: Gets an artist by name.
**Parameters**:
- `name`* (path string): Studio name.
- `userId` (query string): Optional. Filter by user id, and attach user data.

## GET /Audio/{itemId}/stream
**Summary**: Gets an audio stream.
**Parameters**:
- `itemId`* (path string): The item id.
- `container` (query string): The audio container.
- `static` (query boolean): Optional. If true, the original file will be streamed statically without any encoding. Use either no url extension or the original file extension. true/false.
- `params` (query string): The streaming parameters.
- `tag` (query string): The tag.
- `deviceProfileId` (query string): Optional. The dlna device profile id to utilize.
- `playSessionId` (query string): The play session id.
- `segmentContainer` (query string): The segment container.
- `segmentLength` (query integer): The segment length.
- `minSegments` (query integer): The minimum number of segments.
- `mediaSourceId` (query string): The media version id, if playing an alternate version.
- `deviceId` (query string): The device id of the client requesting. Used to stop encoding processes when needed.
- `audioCodec` (query string): Optional. Specify an audio codec to encode to, e.g. mp3. If omitted the server will auto-select using the url's extension.
- `enableAutoStreamCopy` (query boolean): Whether or not to allow automatic stream copy if requested values match the original source. Defaults to true.
- `allowVideoStreamCopy` (query boolean): Whether or not to allow copying of the video stream url.
- `allowAudioStreamCopy` (query boolean): Whether or not to allow copying of the audio stream url.
- `breakOnNonKeyFrames` (query boolean): Optional. Whether to break on non key frames.
- `audioSampleRate` (query integer): Optional. Specify a specific audio sample rate, e.g. 44100.
- `maxAudioBitDepth` (query integer): Optional. The maximum audio bit depth.
- `audioBitRate` (query integer): Optional. Specify an audio bitrate to encode to, e.g. 128000. If omitted this will be left to encoder defaults.
- `audioChannels` (query integer): Optional. Specify a specific number of audio channels to encode to, e.g. 2.
- `maxAudioChannels` (query integer): Optional. Specify a maximum number of audio channels to encode to, e.g. 2.
- `profile` (query string): Optional. Specify a specific an encoder profile (varies by encoder), e.g. main, baseline, high.
- `level` (query string): Optional. Specify a level for the encoder profile (varies by encoder), e.g. 3, 3.1.
- `framerate` (query number): Optional. A specific video framerate to encode to, e.g. 23.976. Generally this should be omitted unless the device has specific requirements.
- `maxFramerate` (query number): Optional. A specific maximum video framerate to encode to, e.g. 23.976. Generally this should be omitted unless the device has specific requirements.
- `copyTimestamps` (query boolean): Whether or not to copy timestamps when transcoding with an offset. Defaults to false.
- `startTimeTicks` (query integer): Optional. Specify a starting offset, in ticks. 1 tick = 10000 ms.
- `width` (query integer): Optional. The fixed horizontal resolution of the encoded video.
- `height` (query integer): Optional. The fixed vertical resolution of the encoded video.
- `videoBitRate` (query integer): Optional. Specify a video bitrate to encode to, e.g. 500000. If omitted this will be left to encoder defaults.
- `subtitleStreamIndex` (query integer): Optional. The index of the subtitle stream to use. If omitted no subtitles will be used.
- `subtitleMethod` (query ): Optional. Specify the subtitle delivery method.
- `maxRefFrames` (query integer): Optional.
- `maxVideoBitDepth` (query integer): Optional. The maximum video bit depth.
- `requireAvc` (query boolean): Optional. Whether to require avc.
- `deInterlace` (query boolean): Optional. Whether to deinterlace the video.
- `requireNonAnamorphic` (query boolean): Optional. Whether to require a non anamorphic stream.
- `transcodingMaxAudioChannels` (query integer): Optional. The maximum number of audio channels to transcode.
- `cpuCoreLimit` (query integer): Optional. The limit of how many cpu cores to use.
- `liveStreamId` (query string): The live stream id.
- `enableMpegtsM2TsMode` (query boolean): Optional. Whether to enable the MpegtsM2Ts mode.
- `videoCodec` (query string): Optional. Specify a video codec to encode to, e.g. h264. If omitted the server will auto-select using the url's extension.
- `subtitleCodec` (query string): Optional. Specify a subtitle codec to encode to.
- `transcodeReasons` (query string): Optional. The transcoding reason.
- `audioStreamIndex` (query integer): Optional. The index of the audio stream to use. If omitted the first audio stream will be used.
- `videoStreamIndex` (query integer): Optional. The index of the video stream to use. If omitted the first video stream will be used.
- `context` (query ): Optional. The MediaBrowser.Model.Dlna.EncodingContext.
- `streamOptions` (query object): Optional. The streaming options.
- `enableAudioVbrEncoding` (query boolean): Optional. Whether to enable Audio Encoding.

## GET /Channels/{channelId}/Items
**Summary**: Get channel items.
**Parameters**:
- `channelId`* (path string): Channel Id.
- `folderId` (query string): Optional. Folder Id.
- `userId` (query string): Optional. User Id.
- `startIndex` (query integer): Optional. The record index to start at. All items with a lower index will be dropped from the results.
- `limit` (query integer): Optional. The maximum number of records to return.
- `sortOrder` (query array): Optional. Sort Order - Ascending,Descending.
